# Zypher V1

`V1` becomes the clean public baseline for Zypher: the latest desktop app state, a polished repo presentation, and a custom Windows installer as the primary release asset.

## Release Highlights

- Ships the newest app build, including the fullscreen title bar fix already present locally.
- Promotes the custom Windows installer to the main release path.
- Keeps GitHub-generated source archives alongside the installer for every release.
- Refreshes the public docs so the repository, install flow, and project scope are easier to understand at a glance.

## Included In This Release

- Rich text journaling with headings, quotes, code blocks, tables, inline images, and slash commands.
- Library organization with folders, tags, pinning, search, sort modes, and quick switching.
- Recovery features including Trash, saved page history, manual backups, and automatic backup generation.
- Export and import support for JSON, encrypted JSON, encrypted ZIP full-app backups, HTML, TXT, and Markdown.
- Local privacy controls including password protection, idle lock, encrypted storage, cooldowns, and optional self-destruct on repeated failed unlock attempts.
- Reflection tools including "On This Day" memories and monthly or yearly review generation windows.

## Downloads

- Windows installer: `Zypher-Setup-V1-Custom.exe`
- Source code: use the GitHub source archives automatically attached to the release

## Installer Notes

- The custom installer supports install, update, repair, and uninstall flows.
- Uninstall removes the app binaries and shortcuts but leaves local journal data in your Windows profile.
- The release is unsigned, so Windows may show an "Unknown publisher" warning before installation.
